Dynamic Programming


// Fibonacci using DP (Memoization)
int fib(int n, vector& memo) {
    if (n <= 1) return n;
    if (memo[n] != -1) return memo[n];
    memo[n] = fib(n-1, memo) + fib(n-2, memo);
    return memo[n];
}

// Time Complexity: O(n)
// Space Complexity: O(n)

Divide and Conquer


// Merge Sort Implementation
void mergeSort(int arr[], int l, int r) {
    if (l < r) {
        int m = l + (r - l) / 2;
        mergeSort(arr, l, m);
        mergeSort(arr, m+1, r);
        merge(arr, l, m, r);
    }
}

// Time Complexity: O(n log n) in all cases
// Space Complexity: O(n)

Quick Sort Algorithm


# Quick Sort Implementation
def quickSort(arr, low, high):
    if low < high:
        pivot_index = partition(arr, low, high)
        quickSort(arr, low, pivot_index - 1)
        quickSort(arr, pivot_index + 1, high)

def partition(arr, low, high):
    pivot = arr[high]
    i = low - 1
    for j in range(low, high):
        if arr[j] <= pivot:
            i += 1
            arr[i], arr[j] = arr[j], arr[i]
    arr[i + 1], arr[high] = arr[high], arr[i + 1]
    return i + 1

# Space Complexity: O(log n)

Why Quick Sort for E-commerce?

  • Efficiency: O(n log n) average time complexity for large data sets
  • Space Efficient: Uses in-place sorting, reducing memory overhead
  • Scalability: Efficient for massive product catalogs
  • Flexibility: Can be customized for sorting by different attributes (price, rating, etc.)
  • Performance: Improves sorting time and user experience in product discovery
